# Sonic Robo Blast 2

[Sonic Robo Blast 2](https://srb2.org/) is a 3D Sonic the Hedgehog fangame based on a modified version of [Doom Legacy](http://doomlegacy.sourceforge.net/).

## Compiling

Please refer to the wiki.

### Requirements
- Gradle
- Java Development Kit (JDK)
- Android SDK
